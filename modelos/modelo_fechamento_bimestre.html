♿ ADAPTAÇÕES ESPECÍFICAS BIMESTRE 1 (Caos Acessível para Todos)
📌 Para TEA (Transtorno do Espectro Autista)
  • Use checklist visual com etapas (1. Ouvir, 2. Repetir, 3. Colar).
  • Prefira ritmos musicais previsíveis para os loops (evite barulhos abruptos).
  • Na caça ao tesouro (semana 5), ofereça um mapa simplificado e sem obstáculos surpresa.
  • Permita que o aluno use abafador de ruído se necessário.
📌 Para TDAH (Déficit de Atenção)
  • Timer explosivo: use cronômetro de 5-7 minutos para cada bloco e um sino estridente para marcar a troca.
  • Hiperfoco recompensado: o aluno que terminar o algoritmo primeiro pode ser o "líder do loop" do grupo.
  • Divida as atividades em partes curtas com recompensas imediatas (adesivo a cada etapa concluída).
📌 Para Deficiência Física
  • Comando por voz: o aluno pode ser o "programador-mor" e ditar os comandos `repita` para os colegas executarem.
  • Para o labirinto humano (semana 2), o aluno pode ser o "leitor do algoritmo" em vez de andar.
  • Use robôs de chão (tipo Beebot) em um tapete adaptado para alunos com limitação motora nos membros superiores.
📌 Para Altas Habilidades
  • Loopception (Loop aninhado): Desafie-os a criar `repita 3 vezes { repita 2 vezes { pular } }`.
  • Tutor de robótica: Peça que criem um tutorial em vídeo explicando o conceito de loop para alunos do 2º ano.
  • Desafio extra: criar um algoritmo que desenha um polígono de 12 lados (calcular o ângulo sozinho).

A Corrida dos Comandos Repetidos - Monte seu algoritmo com cartões!

LOOP DASH - A CORRIDA DOS COMANDOS REPETIDOS

🤖 Contexto: Loop Dash é um jogo onde você controla um robozinho que só pode se mover usando comandos de REPITA. O objetivo é chegar ao final da pista usando o MENOR número de cartões possível!

🎯 Público-alvo: Alunos do 3º ano (individual ou duplas)

💡 Conceito estrela: Loop como otimização de código (menos cartões = mais eficiência)

📜 Regras do Jogo (Versão CARTOONS - sem digitar!):
  1. 🏁 Pista de 3 fases: RETA → ZIGUE-ZAGUE → OBSTÁCULOS
  2. 🎴 Monte seu algoritmo: Clique nos cartões de comando para construir a sequência
  3. 🔁 Regra de ouro: Use o cartão REPITA para repetir ações - NADA de comandos soltos!
  4. 🏆 Pontuação: Quem usar MENOS cartões vence! (Menos linhas = mais pontos)
  5. 🐛 Bug e penalidade: Se o robô bater num obstáculo, você tem que DEPURAR (trocar cartões)
  6. 🌟 Bônus criativo: Usar um REPITA dentro de outro = título "Mestre do Loopception" + estrela dourada!
